# 素数环

## 题目要求

对于一个给定的整数n，将数字1到n排成一个环。

第1个数字和第2个数字相邻。

第2个数字和第3个数字相邻。

第i个数字和第i+1个数字相邻。

特殊的，第n个数字还和第1个数字相邻。

如果任意两个相邻的数字之和都是素数，我们称这个数字环为素数环。

## 输入格式

输入一个数字n。

## 输出格式

依次输出所有方案。每一行n个用空格分隔开的数字，表示一个方案。

## 样例输入

```
5
```

## 样例输出

```
1 4 3 2 5
3 4 1 2 5
5 2 1 4 3
5 2 3 4 1
```

## 数据范围

n大概是不超过20的……
